HTML: stands for Hyper Text Markup Language, it's the standard markup language for creating Web pages,... it also describes the structure of a Web page, consists of a series of elements these elements tell the browser how to display the content. The elements label pieces of content such as "this is a heading", "this is a paragraph", "this is a link", etc.
CSS: Cascading Style Sheets, fondly referred to as CSS, is a simply designed language intended to simplify the process of making web pages presentable. IT allows you to apply styles to web pages. More importantly, CSS enables you to do this independent of the HTML that makes up each web page. It describes how a webpage should look: it prescribes colors, fonts, spacing, and much more. In short, you can make your website look however you want.
